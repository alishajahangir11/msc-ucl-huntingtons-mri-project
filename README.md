# MRI Biomarkers in Huntington’s Disease

## Overview

This project explores MRI-based biomarkers in Huntington’s disease using longitudinal neuroimaging data.

The aim is to identify non-striatal brain regions that could serve as reliable biomarkers in gene therapy trials, where traditional striatal markers may be affected by surgical intervention.

## What I Did

* Analysed MRI data from HD-YAS study
* Compared HD gene-expanded vs controls
* Calculated atrophy rates across the whole brain
* Ranked regions using effect size (Cohen’s d)
* Studied relationships with CAP100 and disease stage

## Key Findings

* Caudate and putamen showed strongest atrophy
* Non-striatal regions also showed significant changes
* These may be useful biomarkers in future clinical trials
